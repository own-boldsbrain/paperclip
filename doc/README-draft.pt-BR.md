# O que é

Paperclip é um servidor e interface do usuário para coordenar uma equipe de agentes para executar negócios.

Gerencie seus próprios agentes, defina metas e acompanhe o trabalho e os custos dos seus agentes.
É semelhante a um gerenciador de tarefas, mas possui habilidades especializadas e coordenação de agentes.

**É elegante e fácil de usar**

# Paperclip é ideal para você se:

- Você deseja construir empresas autônomas
- Você deseja coordenar vários tipos de agentes (OpenClaw, Codex, Claude, Cursor, etc.) para colaborar em direção a um objetivo comum
- Você tem 20 terminais Claude Code abertos simultaneamente (e perde o controle do que todos estão fazendo)
- Você deseja permitir que seus agentes operem autonomamente 24 horas por dia, 7 dias por semana, mas ainda assim audite seu trabalho e intervenha quando necessário
- Você deseja monitorar os custos
- Você deseja um processo para gerenciar seus agentes que se assemelhe a usar um gerenciador de tarefas
- Você deseja gerenciar suas empresas autônomas do seu telefone

# O que não é

Paperclip não é:

- Uma ferramenta de revisão de código
- Uma ferramenta opinativa sobre qual agente você usa
